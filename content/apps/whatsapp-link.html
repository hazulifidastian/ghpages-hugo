Membuat link WhatsApp Me dengan menggabungkan api WhatsApp yang digunakan untuk menginisiasi pengiriman pesan melalui browser, kemudian nomor telepon tujuan dan pesan yang akan dikirimkan.

Aplikasi ini dibuat reaktif menggunakan AlpineJs1. Otomatis memantau perubahan pada input nomor telepon dan input pesan kemudian mengkompilasi kedua input tersebut menjadi link.

Salin teks diatas untuk dijadikan url.
